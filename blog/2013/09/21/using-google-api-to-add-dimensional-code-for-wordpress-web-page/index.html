Using Google API to Add Dimensional Code for WordPress Web Page

Sep 21st, 2013 | Comments

Smart phones becoming increasingly popular, website support for mobile phones is becoming more important, two-dimensional code can be said to be the most simple and convenient way to deliver the message to the mobile phone, especially transport web URL. Such http://www.mrxuri.com/, generated for the dimensional code image URL is https://chart.googleapis.com/chart?cht=qr&chs=220%C3%97220&choe=UTF-8&chld=L|2&chl=http://www.mrxuri.com/

https://chart.googleapis.com/chart? is a dimensional code API address.

cht = qr indicate the type of two-dimensional code.

chs = 220 Ã— 220 generated two-dimensional code image size.

choe = UTF-8 encoding of the content, the default is UTF-8.

chld = L|2 error correction levels and blank border width. Error correction level from low to high, respectively LMQH, the higher the level, the lower the error rate of the rows and columns of dimensional code is also more (Google API based on dimensional code and it will automatically select the appropriate content-length two-dimensional code ranks number) (Google API documentation in more detail). Border width can choose.

chl = http://www.mrxuri.com/ contents of dimensional code.
